# Designsystem

## Stilrichtung

Minimalistisch.
Leicht gerundet.
Klar.
Ohne dekorative Software-Effekte.

## Regeln

- Buttons, Inputs, Selects und Icon-Buttons bekommen einen 2px Inset Shadow.
- Sonst möglichst keine Drop Shadows.
- Leichte Border-Radius.
- Bauhaus-Farben als einzige farbige Akzente.

## Farben

- Bauhaus Blue: `#0057B8`
- Bauhaus Red: `#E33A2C`
- Bauhaus Yellow: `#F2C500`

Neutrale Farben:
- Hintergrund: `#F7F7F4`
- Surface: `#FFFFFF`
- Text: `#111111`
- Border: `#D9D5CD`
- Muted Text: `#6C6A66`

## Radius

- Buttons: 10px
- Inputs: 10px
- Cards: 12px
- Modals: 16px

## Shadow

```css
--inset-shadow: inset 0 0 0 2px rgba(0,0,0,0.06);
```

## Buttons

Primary:
- blau
- weißer Text
- inset shadow

Secondary:
- helle Fläche
- dunkler Text
- inset shadow

Danger:
- rot
- weißer Text
- inset shadow

## Cards

- 1px Border
- keine schweren Schatten
- nur sehr subtiler Hover

## Icon Buttons auf Library-Karten

- klein
- leicht gerundet
- inset shadow
- bei Hover sichtbar
- klarer Kontrast

## Typografie

Empfehlung:
- Inter oder Geist
- sachlich, kein Display-Font

## Spacing

8px-System:
8 / 12 / 16 / 24 / 32 / 48
