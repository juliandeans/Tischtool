# Library, Versioning, Export und Kosten

## Library

Die Library ist das Arbeitsgedächtnis des Produkts.

Sie muss können:
- Bilder anzeigen
- nach Projekt filtern
- nach Datum sortieren
- editieren
- herunterladen

## ImageCard

In jeder Karte:
- Thumbnail
- Name
- Projekt
- Zeit

Hover rechts oben:
- Download
- Edit

## Download

Für den MVP:
- PNG-Download

Später:
- JPG / WebP / ZIP-Projektexport

## Edit

Edit öffnet den Editor mit dem aktuellen Bild als Ausgangspunkt.
Neue Bearbeitungen überschreiben nichts, sondern erzeugen neue Kind-Bilder.

## Kosten-Tab

Oben:
- heute
- Monat
- Durchschnitt pro Bild
- teuerstes Projekt

Darunter:
- Tabelle aller Requests
- Diagramm Kosten pro Tag
- Diagramm Kosten pro Projekt

## Warum Kosten ein eigener Bereich sind

Bild-Requests kosten einzeln oft nur Centbeträge, summieren sich aber durch Varianten und Iterationen. Ein sichtbarer Kostenbereich macht das Produkt kontrollierbarer.
